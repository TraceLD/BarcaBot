Available commands

The bot is actively maintained, so new commands can be added at any time. If you have an idea for a command, contact Trace#8994 on Discord or open a GitHub issue.

Player info

=player <player_name>
Shows information (including statistics) about a Barca player.

FC Barcelona matches

=schedule
Shows upcoming Barca matches.

LaLiga Table

=table top
Shows LaLiga's current Top 5.

=table bottom
Shows LaLiga's current Bottom 5.

LaLiga Top Scorers

=scorers
Shows LaLiga's current top scorers.

Statistics

=stats <player_name>
Shows statistics chart for a given player.

=stats <player_name> <player_name2> <player_name3> etc.
Shows statistics chart comparing given players.

Data source

BarcaBot requires a lot of data to function, from player stats to the match schedule. I've put a lot of thought into where this data should be obtained from. The easiest (and cheapest) solution would be to scrape sites like WhoScored, Flashscores, etc. using one of the scraping libraries as that would provide access to the largest array of data, however, this solution comes with many legal, as well as ethical conerns, and therefore I've deemed it unsuitable. There are football stats and data provides such as Opta, but the issue with them is that they can cost hundreds, if not thousands of pounds per month.

I've ended up going with a combination of two Freemium APIs - API-FOOTBALL and FootballData. They both provide a basic free tier, which I can use for everything BarcaBot needs. Instead of calling the API every single time a user requests a command, I'm updating my own database with data on a schedule. This, while not providing truly real time data, allows me to essentially bypass any rate limits.
