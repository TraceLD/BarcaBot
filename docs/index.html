Available commands

The bot is actively maintained, so new commands can be added at any time. If you have an idea for a command, open a GitHub issue or contact Trace#8994 on Discord.

Team information

/teamstats

Shows information about the FC Barcelona team including form and statistics (such as goals scored/conceded).

Player information

/player <player_name>

Shows information and statistics for a FC Barcelona player.

FC Barcelona matches

/matches Last

Shows results of the last five FC Barcelona matches.

/matches Upcoming

Shows information about the next five FC Barcelona matches.

LaLiga Top Scorers (Pichichi)

/pichichi

Shows the top scorers of LaLiga Santander. The top scorer at the end of the season will be awarded the Pichichi Trophy.

Statistics charts

/stats <player_name>

Shows statistics chart for a given player.

/stats <player_name> <player_name2>

Shows statistics chart comparing given players.

Miscellaneous

/about

Shows information about BarcaBot.

/invite

Shows an invite link for BarcaBot.
